



Start Ganache:

Truffle Migrate:

Navigate to your Truffle project directory in command prompt.
Run "truffle migrate --reset" to deploy your contracts onto the Ganache blockchain.

copy contract address from console and paste in app.js

Start local HTTP Server:

Navigate to your frontend project directory in the terminal.

"http-server"


Open your Web Application:

Go to your browser and visit http://localhost:8080 (or whatever port you chose).

Connect MetaMask:

Open MetaMask extension in your browser.
Connect to the Ganache network as previously set up.
Make sure you're logged into one of the Ganache accounts or an account that you've imported from Ganache.

Interact with the Web Application:

Use your web application's interface to interact with the smart contract.
Actions performed on the web application should be shown on the Ganache blockchain
